传染病和梦想

October 13, 2004 11:50 AM | 0 Comments

今天司机出差,只好很郁闷的乘地铁上班.在地铁里咳了两声,坐在我对面的小姑娘赶紧挥了挥手,好像要把细菌赶走.我心里暗笑:赶得走吗?有故意冲着她咳了两下,谁让你不乘自己的车呢?我也不想上班呢,谁让没得传染病呢.

走到美罗城,居然还要排15米以上的队上电梯.我惊天地泣鬼神的咳了一阵,眼泪都快咳出来了,熟人和生人都看着我,好像我很垂危的样子.不管,照样咳嗽.不是我愿意的,是忍不住.想想明天要和那么多人开会,只盼他们不要被我吓倒.

打开信箱,居然有人在昨天晚上半夜12:30发了封信给我,催要一封Proposal.真想回他一封信说，你找的人生病，不治身亡。

决定今天好好修养，和一个朋友聊天。

Wendy Fan - 我好想得传染病 说：
你还有梦想么, J?
J 说：
我的梦想是
J 说：
在一个有花园的房子里晒太阳
J 说：
看小说
J 说：
喝茶
J 说：

Wendy Fan - 我好想得传染病 说：
但是你的房子好像没花园呀。
J 说：
是啊
J 说：
所以以后再买呀
Wendy Fan - 我好想得传染病 说：
还买？
J 说：
嗯
J 说：
再买一个
Wendy Fan - 我好想得传染病 说：
你已经两套了，累不累？
J 说：
等以后我有钱了
J 说：
呵呵
Wendy Fan - 我好想得传染病 说：
还欠100多万呢。
J 说：
所以说是梦想阿
J 说：

J 说：
你呢
J 说：
梦想是什么
Wendy Fan - 我好想得传染病 说：
当经理
J 说：
Wendy Fan - 我好想得传染病 说：
开玩笑
Wendy Fan - 我好想得传染病 说：

Wendy Fan - 我好想得传染病 说：
近两年的愿望是不要被fire
Wendy Fan - 我好想得传染病 说：
过两年的愿望是早点被fire
Wendy Fan - 我好想得传染病 说：
赔我一大笔钱
Wendy Fan - 我好想得传染病 说：
然后让我可以周游列国
J 说：
不够
Wendy Fan - 我好想得传染病 说：
赔多少?
J 说：
等他们赚了钱就行了
J 说：
做几年就赔几个月
Wendy Fan - 我好想得传染病 说：
钱是永远不够的，他们赚那2，30十万，够干啥？
J 说：
周游列国肯定够了
Wendy Fan - 我好想得传染病 说：
那时候他们一定很忙，没空陪我们
J 说：
那我们两个去好了
Wendy Fan - 我好想得传染病 说：
回来他们全包二奶了
J 说：
那我就休了他
J 说：
瓦嘎嘎嘎
J 说：
然后分一半钱给我
J 说：
又可以出去玩了
Wendy Fan - 我好想得传染病 说：
我还是有一些同情心，觉得让他们工作，我们出去玩，不大好
J 说：
哪有什么
J 说：
他们辛苦赚钱不是给我们用的吗？
J 说：
要是我要出去玩，G肯定很高兴
J 说：
终于走掉了，可以专心工作了
J 说：
哈哈
Wendy Fan - 我好想得传染病 说：
是不是你不会烦他了。他可以专心工作了
J 说：
是啊
J 说：

其实我的要求还是很简单，也不是很懒，很想工作，但希望有一些时间上的自由和工作上的乐趣。比如说，生病的时候可以在家工作，而不用提心吊胆的担心老板会找不到，或者会不满意。
