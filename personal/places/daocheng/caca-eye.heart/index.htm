掠过眼睛，藏入心底
                                                              ----游说稻城

文：卞卡 图：王建硕

[ 第一部分：在成都 ] [ 第二部分：成都-新都桥 ] [ 第三部分：日瓦-亚丁 ] [ 第四部分：亚丁-成都 ] [ 第五部分：致谢 ]

从做出决定到出发前往稻城，只有一个星期的时间。有朋友说实在是有点仓促，可工作的原因让我无法提前安排。再加上自己也喜欢这样的心血来潮，于是说服自己说服同伴，前往这个向往已久的“香格里拉”。

9月28日，假期的第一天，早上坐了三个小时的飞机来到成都，这个有着天府之国美誉的西南名城。把自己安顿在人民路北三段的地矿招待所后，就出来边逛边找吃的。除了有点气势的天府广场、满大街的老字号餐馆以及为数不少的“烂尾楼”之外，成都没有给我留下太多新奇的印象。与沿海地区相比，这里的空气让人感到了一点闭塞，现代化的程度有点落后。不过成都人生活地实在是悠闲，走路是慢悠悠地，一大清早就摆开麻将桌，吃饭时间随便走进一家餐馆或是茶楼都是人满为患……怪不得当地的出租车司机都说成都人“好吃懒做”，而且浙江人“很坏”，因为“钱都被他们赚走了”！

吃午饭的时候已经是下午三点了，找的地方是很有名的“陈麻婆豆腐”。很喜欢他们的店堂风格，旧旧的酱色木桌椅，雕花的木窗子，平易近人的风格飘着麻辣的香味。不过不太敢吃麻婆豆腐，以前领教过这等辣味，怕了。点了几样不淡不辣的，最好吃的是酱爆鸭肉。

下午5点的时候去了武侯祠，一个纪念诸葛亮的园林。大殿两边排列了文官武将的塑像，武将的名字都很熟，文官却打多不认得。可能跟我没有好好看完三国有关吧。曾经看过苏州的园林，杭州的园林，北京的园林，说实话，除了不同的文化底蕴，这个武侯祠真的不怎么样。

晚上原本打算去看望江楼，后来被同行的伙伴拉去了川大见她表妹。川大实在是大，光是一幢行政楼就让我大开了眼界。她们还带着我逛啊逛，从南到北从北到西，终于逛到一个好玩好吃的地方。那条路据说叫文化路，小小的街道两边开满了服装店、饰品店和“串串香”。刚开始的时候还没怎么在意，后来听说那些摆满了玻璃橱的火锅串才一毛钱一串，我立刻就来劲了！于是几个人招呼着来到一家客人最多的“串串香”，叫了鸳鸯锅，每人抓了一大把自己喜欢吃的各种烫料，蔬菜、牛羊肉、鸡鸭肠应有尽有，尽管分量比较袖珍，但一毛钱的价格还是让人觉得实在是便宜！两个小时之后，谁也吃不下了。叫来服务员一数签子，160根，就是16块钱。加上汤底调料，一共是30块钱，我们几个人都大叫便宜！

望江楼去不成了，不过这个晚上过地也挺精彩，回到住处躺下睡觉时，已将近12点了。
