掠过眼睛，藏入心底
                                                              ----游说稻城

文：卞卡 图：王建硕

[ 第一部分：在成都 ] [ 第二部分：成都-新都桥 ] [ 第三部分：日瓦-亚丁 ] [ 第四部分：亚丁-成都 ] [ 第五部分：致谢 ]

中午的时候到了日瓦乡，这是进入亚丁景区的最后一个补给站。

我们的车子开近绿野社区的时候，我看见左边的藏家酒楼的二楼阳台上有三个远行的背包族坐在地上休息。午后的阳光印在他们脚边，年轻的脸上一副自在自信的神情。那一刻，我觉得自己看到的是三个把青春和汗水洒在旅途上的年轻的灵魂，他们的心飞翔在高空之上，他们的眼睛只为身边的山林睁开，他们脚步踩着坚实的土地，却永远没有羁绊，没有停留。那一刻，我看他们是仰视，可我觉得他们比身边的很多人更清晰，更真实。

绿野社区的门口墙边，有一群藏族妇女小孩，一位年长的老人坐在中间。不知该如何确切描述她的神态，岁月在她脸上留下了沧桑，也留下了安详。她手里握了一串佛珠，嘴里不停地喃喃念经。一条天蓝色的围巾，赭色的棉袍和红色的绣边，尽管色彩鲜艳，可穿戴在她的身上，却一点都不显得突兀，浓墨重彩反而让她更加精神起来。大家纷纷和她合影留念。

绿野社区的墙上挂满了留言签字的红布，印象最深的是一幅太阳，光芒是由手印组成的，然后在手印上签上名字，太阳中间的圆盘上画了笑脸，很有创意。我们也扯了块红布留下自己的大名，最有意思的是法官的签名，一匹马儿跑在草地上。现在，他真的永远跑在亚丁的草原上了。

在日瓦小憩之后，继续上路，迫切的心情让大家都有点无心留意身边的风景了。在半山腰转了个弯，仙乃日就在眼前安然地出现了。车厢里自然是一阵欢呼惊叫，此时遇到前方修路，大家下车步行一百米，这一百米可不知谋杀了多少菲林！回到车上，大家更兴奋了，唧唧歪歪地拉着香格里拉山庄的王经理问这问那，没多久就到了目的地――念青贡嘎日松贡布（亚丁）。

从山上看亚丁村的时候，令人想起某本杂志上的照片。高原阳光下的小村落散布在山腰一片肥美的草场之上。高山石垒起的藏族民居绘着彩色的窗子和屋檐，黑色的牦牛在草场上漫步，一条条小路交错成和谐的几何图案。抬起头，就是仙乃日雪山。

亚丁景区有三座雪山，分别是仙乃日（观音菩萨雪山）、央迈勇（文殊菩萨雪山）和夏诺多季（金刚菩萨雪山），它们成品字型排列，洛绒牛场就在三座雪山之间。不过雪山上云雾缭绕，很少有机会一起看到三座雪山露出尊容。*

经过亚丁村来到山谷，面前一条小小的马道就是通往亚丁主景区的冲古寺、珍珠海、洛绒牛场和牛奶海五色海的必经之路。从这里上去可以到达海拔5000米以上。

晚上我们住在一间木屋里。吃过晚饭，在法官和晏琳的邀请下，大家围在一起玩起了杀人游戏。那天晚上真的很尽兴，以至于我笑得头开是隐隐作痛，估计是消耗了不少体力。可在场的所有人都没有想到，厄运正在悄悄逼近。

当天晚上下了场大雪，到第四天早晨，附近的山上都银装素裹了。考虑到自己的身体状况和高原经验，我们选择了骑马上山。带领我们山上的是当地导游多吉，他的名字就是金刚雪山的意思。我挑了一匹枣红色的大马，除了马之外就是骡子。第一次把马和骡子放在一起比较，原来骡子耳朵大身体大，脸看上去有点像驴，毛皮基本上都是深色的，而且脾气不好，没有马听话。给我牵马的马夫叫尔折，看上去还不错。从龙龙坝出发，经过一座小木桥，就开始在进山的马道上前行了。

刚开始的时候，团友们都走在一起，因为基本上都是初次骑马，所以显得有点兴奋，身边的景色也顾不上多看了。在一个坡道前有两条路，大家都走了一条比较近的，只有法官绕了远路，后面的人都觉得有点奇怪，马夫说那边坡度小一点。现在想起那时候的情景，发觉法官的状态真地好象不是太好，感觉有点焉焉的。可谁又想得到，他绕远路是怕自己上坡的时候坐不住呢？那时候他一定已经很虚弱了。

马道非常泥泞，两边原始的松柏林散发着怡人的清香，一条小溪沿山谷潺潺而下，向前望去，是云雾缭绕的山麓。坐在马背上，大家都兴奋不已，谈笑风生，还不停地拍照。一小时后，到达了冲古寺。

冲古寺这个词常被指代为亚丁自然保护区的中心景区，因为那座小小的喇嘛庙是这里的标志物。冲古寺下的那块草甸，有蜿蜒的小河流过。那些粗壮的杨树，连接伸展成一片郁郁葱葱的景致。踩踏在软软的草甸上，远远地望见央迈勇的尖顶，还有夏洛多吉隐藏在云雾中羞涩的脸庞。

从冲古寺再往前半小时，就到达了洛绒牛场。一片肥美的草场，正对面就是央迈勇，左边是夏诺多季，右边仙乃日。可惜的是，夏诺多季和仙乃日都被云雾遮挡，央迈勇也只小露了一下尖顶。不过洛绒牛场实在是美，在雪山的包围下，有小溪静静地流过，彩色的灌木林沿坡而下，午后的阳光洒在人们脸上，心情马上就开朗起来。在往前就是神秘美丽的五色海和牛奶海了，很想去看看，可是在多吉的劝阻下，还是放弃了。现在想想有点后悔，不去试怎么知道不行呢？

从洛绒牛场回到冲古寺，转入一条上山的小路来到珍珠海（卓姆拉措），一个由仙乃日的融水形成的高山湖泊。来到仙乃日脚下，可以看到一片长满灌木的平地，左边有一条小河，河底是清一色的白色石子。

不过如果你足够细心，你会发现白色石子中夹杂着红色、绿色、黑色、黄色的彩石，在清泠的水底，闪烁着剔透的光芒。穿过右边高大的树林，在密林深处碧绿的湖水清澈无比，映照着仙乃日的身影，微风吹来，波光粼粼，四周山上的冷杉林层叠翡翠，真有飘飘欲仙之感。

在半山腰上，冲古寺面对仙乃日而坐。几间石头房子，一道幽暗的小木梯，有活佛为你摸顶祝福，有虔诚静谧的氛围笼罩你。在冲古寺里徘徊的十几分钟里，我屏住了呼吸，不敢大声说话，生怕惊扰了仙人，也不愿破坏这庄严肃穆的时刻。在冲古寺后的一小块空地上，是徒步者的营地，在那里还可以远远地望见亚丁村，那个仿佛不食人间烟火又飘着炊烟的小小村落。

回来的路上，看到法官坐在草地上托着头，跟他打招呼也没有回答。当时还在心里发笑：一个大男人怎么这么弱啊！现在想来，真是追悔莫及，如果那时候就意识到点什么就好了。

晚上的篝火晚会真是过瘾，当我靠在那张刚剥下来的羊皮上拍照的时候如果低头看一眼，拍出来的肯定是另一种效果！亚丁的夜晚好冷，靠近火堆怕被烟熏着，远离火堆又会牙齿打架，不过后来大家一起手拉手跳圆圈鞠躬舞倒还不错，尽管姿势别扭，热身效果很好。特别怀念围着火堆唱歌的那会儿，有人带头就有人应和，那歌声就飘过头顶，飘过山梁，飘过雪峰，直到天外了……

快到十一点钟的时候下起了小雨，羊也烤熟了，尽管烤羊的味道有点“原始”而且不是很熟，可我觉得很好吃，还直舔手指。吃过烤羊之后，好像一天的事情才算完结，于是陆续进帐篷睡觉。我们的帐篷是十六人的大帐篷，里面铺着两排通铺，我还从来没有睡过条件这么差的“馆”呢，不过这不算什么，反而更有野外的意味。

落叶红一个人坐在篝火的余烬边若有所思，全然没有了刚才又唱又跳的兴奋。在这样的夜晚安静地坐一会儿，也真是一种享受，可以理清头脑中烦乱的思绪，或许还能痛下决心，做出某个重要的决定呢！

半夜里被风雨声惊醒了。我睡在门边，布帘子不遮雨，鞋子和包都被雨打湿了，帐篷的边边角角还有点漏水。赶紧起来拿出防水布遮住鞋包，才又安心地睡下。

第二天清早，也就是整个亚丁之行的第五天，7点半左右醒来，便急着要出去拍雪山。想象中那仙乃日在朝阳的映照下一定是金光闪闪，如火炬般伸向蓝天的。拿着相机出门一看，果然不错，来不及刷牙洗脸就要拍照。此时，帐篷里有人喊了起来，来不及按下快门就跑了回去，于是，我经历了一件终生难忘的事情……

我不想再去描述那天的情景那天的心境，尽管那每一分钟都深深地印刻在我的脑海里。我只是惋惜，只是痛心，或许是亚丁的神山也爱他的自由之心吧，舍不得他走就把他留下了……

然后是一路的沉默。回稻城的路上，遇到了微软先生，晚上，和导游会合在稻城。原计划安排的泡茹布查卡温泉，也因为心情和时间的原因不再让人感到那么期待了。再加上安排不周，于是就在县城里找了家浴室随便冲了一下，毕竟已经5天没有洗澡了。
