PVG Schedule Spec

Vision:

The accurate and usable source of flight information to and from Shanghai.

Database format

The proprietary log is pipe-delimited and new line separated. Each record contains null leading and trailing fields. The fields are:

0 NULL
1 flight #
2 source airport code (three letters)
3 destination airport code (three letters)
4 scheduled departure time
5 scheduled arrival time
6 actual departure time
7 actual arrival time
8 ticket island
9 memo
10 NULL/newline

Here are the sample of the database

|CZ9516|SHA|XNN|17:45|21:05|||011||
|UA858|SHA|SFO|13:00|23:25|13:18||||
|LH729|PVG|FRA|13:10|00:30|13:36|00:28|021||

The data is stored in the files. One file for one day. The file name should be 20030212.txt
