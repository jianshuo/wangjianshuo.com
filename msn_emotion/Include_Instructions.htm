Instructions to add MSN Emotions to your MSN Messenger:

If you only want one or few icons:

  1. Select the emotion icon on the top.
  2. Right click on the icon and choose "Safe Pictures As..." and choose any location to save it to your computer.
  3. Open MSN Messenger
  4. Click on the arrow beside the first icon (the smiley face) and click "My Custom Emoticons".
  5. Click "Add"
  6. Click "Find Image" button and browse to the saved MSN Emotion
  7. If you want, you can also assign a keyboard shortcut to the icon.

If you want all icons:

  1. Click the "Download ...zip" link above.
  2. Choose "Save..." when promoted.
  3. Unzip the file with Winzip or other application, and put the files into a folder
  4. Follow steps 3 to 7 above to install these icons.

There is only icons in the zip files. No readme file, no hidden link, no executable files, no virus. Just the icons by themselves.
